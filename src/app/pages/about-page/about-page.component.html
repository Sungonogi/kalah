Kalah and Stickfish

Kalah is a two-player strategy board game, where players distribute stones across pits to capture points. At FAU, we had a Kalah tournament in which my program, Stickfish (written in Java), secured second place. The first-place winner was a C++ program, which inspired me to rewrite Stickfish in C++, making it even stronger.

Tech Stack

This game is built using Angular for the frontend and WebAssembly for Stickfish. The C++ Code of Stickfish is compiled to WebAssembly, meaning the engine runs entirely in your browser without needing a server connection.
